## Vrinda-Store-Sales-Analysis

## Project Overview

The Vrinda Store Sales Analysis project focuses on analyzing retail sales data to uncover trends, customer behavior, and revenue patterns.

The goal of this project is to transform raw transactional data into actionable insights and build a dashboard that helps stakeholders monitor performance and support business decisions.

## Business Problem

* Vrinda Store needed better visibility into:

  - Monthly sales trends

  - Customer demographics

  - Top-performing sales channels

  - State-wise sales performance

  - Order status and fulfillment

*This project provides insights to support data-driven decision-making.*

## Dataset Information

The dataset includes:

* Order details

* Customer demographics

* Sales amount and quantity

* Sales channels

* Product categories

* Order status

* Shipping state and city

# Key Insights

Certain states contribute significantly to total sales.

A few sales channels generate the majority of orders.

Adult customer segments contribute the highest revenue share.

Monthly trends reveal seasonal sales patterns.

# Business Recommendations

Focus marketing campaigns on high-performing states.

Increase stock planning for top-selling categories.

Strengthen engagement on top-performing sales platforms.
